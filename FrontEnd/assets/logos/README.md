# Logos Directory

Store all company logos and branding assets here.

## Files to Add

- **logo.png** - Main company/application logo (recommended: 500x500px)
- **logo-small.png** - Small logo for UI (recommended: 64x64px or 128x128px)
- **logo-white.png** - White version for dark backgrounds (recommended: 500x500px)
- **favicon.ico** - Browser tab icon (recommended: 16x16px, 32x32px, or 64x64px)

## Example Usage

```html
<!-- In index.html header -->
<link rel="icon" type="image/x-icon" href="/assets/logos/favicon.ico">

<!-- In sidebar branding -->
<img src="/assets/logos/logo-small.png" alt="Logo" width="40" height="40">
```

## Tips

- Use PNG format with transparency for best results
- Logos should look good at multiple sizes
- Keep file sizes under 100KB for web optimization
- Use SVG format for scalable logos (no pixelation at any size)
